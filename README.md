
## 🎯 Objetivo

El objetivo de este desafío es ayudar al señor Juan a tomar una decisión estratégica sobre cuál de sus cuatro tiendas
deberia vender para invertir en un nuevo negocio. Para ello se realizó un análisis de datos basado en la información 
de ventas registrada en cada una de las tiendas.

El análisis se realizó utilizando **Python**, **Pandas** y **Matplotlib**

## 🗂️ Datos
Los datos provienen de 4 archivos csv:
-Tienda1
-Tienda2
-Tienda3
-Tienda4
Cada conjunto de datos incluye información sobre productos, categorías, precios, costo de envío, calificación, entre otros.

## 🔎 Análisis realizados

Durante el proyecto se realizaron los siguientes análisis:

- 📈 **Ingresos totales por tienda**
- 🛍️ **Cantidad de productos vendidos por categoría**
- ⭐ **Calificación promedio de los clientes**
- 📦 **Productos más y menos vendidos**
- 🚚 **Costo promedio de envío**
- 📊 **Visualizaciones de datos** para facilitar la interpretación de los resultados

## 📊 Visualizaciones

Se generaron diferentes tipos de gráficos para analizar los datos, entre ellos:

- Gráficos de barras
- Gráficos de líneas

Estas visualizaciones permiten identificar patrones y comparar el desempeño entre las tiendas.

## 📌 Resultados

A partir del análisis de los datos se pudo identificar qué tienda presenta el 
**menor desempeño en comparación con las demás**, considerando factores como ingresos,satisfacción del cliente
y costos de envío. Con base en estos resultados, se genera la recomendación sobre que tienda debe vender el señor Juan.
